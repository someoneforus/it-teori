Bilde-lyd

Farger

HSL

Format: hsl(h, s%, l%) h = hue (fargetone) 0-360 (rød = 0/360, grønn ≈120, blå ≈240) s = metning (saturation) 0-100% l = lyshet (lightness) 0-100% (0 = svart, 50% = «normal», 100% = hvitt) Du kan også legge til alfa

RGBA

Format: rgba(r, g, b, a) r, g, b = 0-255 (rødt, grønt, blått) a = alfa 0-1 (0 = helt gjennomsiktig, 1 = helt ugjennomsiktig)

Linear-gradient

linear-gradient(direction, color1, color2) direction kan være vinkel (45deg)

Komprimering

Komprimering av bilder betyr å redusere filstørrelsen slik at bildet tar mindre plass. Tapsfri komprimering (f.eks. PNG) bevarer all kvalitet, mens tapsbasert komprimering (f.eks. JPEG) fjerner noe informasjon for å gjøre filen mye mindre.

Tegnsett

Et tegnsett bestemmer hvilke tegn som finnes og hvilke tall som representerer dem. Gamle systemer brukte ofte ANSI, som har begrenset støtte for språk og kan vise feil tegn. I dag brukes nesten alltid Unicode, spesielt UTF-8, som kan vise tegn fra nesten alle språk samt symboler og emoji. UTF-8 gjør at tekst vises riktig på tvers av programmer og enheter.

Lyd

Digitalisering av lyd betyr å gjøre lydbølger om til tall. Lyden måles mange ganger per sekund (samplingsfrekvens), og hver måling får en verdi som avrundes (kvantisering). Bit-dybden bestemmer hvor nøyaktige disse verdiene er. Etterpå kan lyden komprimeres for å ta mindre plass, enten tapsfritt (all informasjon beholdes) eller med tap (noe lyd fjernes, f.eks. MP3). Digital lyd er lett å lagre, sende og redigere..
